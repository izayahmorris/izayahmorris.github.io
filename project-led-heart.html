Personalized LED Heart Wall Hang

A carved wooden heart with integrated programmable RGB LED lighting

CNC Machining Arduino 3D Printing Electronics

Project Overview

This project combined traditional woodworking with modern electronics to create a personalized Christmas gift. The design features a carved wooden heart with integrated WS2812B RGB LED lighting controlled by an Arduino microcontroller, allowing for multiple programmable lighting animations.

Completion Date: December 2025

Purpose: Create a unique, personalized gift that demonstrates proficiency in CAD design, CNC machining, 3D printing, and embedded electronics.

Design & Specifications

Dimensions

  • Heart Size: 18" × 20"
  • Material: Ash Wood
  • Depth: 3/4"
  • Mounting Options: Wall mount & stand

Key Features

  • Programmable RGB LED strip (60 pixels/meter)
  • Push-button control for animation selection
  • Multiple continuous rainbow effects
  • 3D printed bracket assembly

CAD Modeling & Design

The initial design was created in OnShape with multiple iterations to refine the heart shape and personalize the design. Key design considerations included:

  • Scaling to fit within available wood plank dimensions
  • Creating an offset bracket system (1.5" inset) for electronics mounting
  • Adding assembly holes for wall mounting rings and stand support
  • Ensuring adequate space for LED strip placement within the heart cutout

Design Challenge

Initial design (20" × 22") exceeded standard wood blank dimensions from suppliers. Solution: Rescaled design to fit 18" × 20" section while preserving aesthetic and personalization elements.

CNC Machining & Fabrication

Design Preparation

Design files were exported as DXF from OnShape and imported into VCarve 12.1.5 for toolpath generation. Vector paths were carefully constructed and closed to ensure continuous carving profiles.

Tool Specifications

V-Bit (Engraving)

Diameter: ¼" shank

Angle: 60°

Pass Depth: 0.1875"

Spindle Speed: 20,000 RPM

Feed Rate: 60 in/min

End Mill (Cutting)

Diameter: ¼"

Flutes: 2

Pass Depth: ⅛"

Spindle Speed: 20,000 RPM

Feed Rate: 240 in/min

Machining Process

  1. Centered and secured ash wood blank (18" × 20") on ShopBot
  2. Made initial shallow passes to verify correct centering
  3. Engraved design details using V-bit
  4. Cut main heart contour using end mill
  5. Separated finished piece from scrap material

3D Printing & Bracket Assembly

A custom mounting bracket was designed in OnShape and 3D printed using FDM technology. The bracket provides:

  • Secure mounting points for LED strip assembly
  • Space for breadboard and Arduino microcontroller
  • Proper alignment for electronics within heart cutout
  • Tolerance optimization for component fit

Design Learning

3D printing the bracket provided valuable insight into tolerancing for assemblies. Initial prints required adjustment to achieve proper fit with electronic components. Final iteration achieved tight tolerance without binding.

Electronics & Arduino Programming

Components

Microcontroller

Arduino (ATmega328P)

LED Strip

WS2812B (60 pixels/meter)

Power Smoothing

1000µF Capacitor

Data Protection

470Ω Resistor

Circuit Design

The circuit was implemented on a breadboard to ensure reliable grounding and power distribution:

  • 1000µF capacitor connected across power supply for smoothing
  • 470Ω resistor on data line for signal protection
  • WS2812B LED strip connected to Arduino GPIO pin
  • Push-button input on Arduino pin 2 for animation control

Features

  • Multiple continuous animation modes (rainbow effects, color sequences)
  • Button-controlled mode cycling
  • Adjustable pixel count for custom patterns
  • Smooth color transitions using Adafruit Neopixel library

Problems & Solutions

Problem 1: Material Dimension Constraints

Initial design (20" × 22") exceeded standard wood plank dimensions available from suppliers.

Solution: Rescaled design to fit 18" × 20" section while preserving aesthetic and personalization elements. Used OnShape's flexible design approach to iterate quickly.

Problem 2: Electrical Grounding Issues

Initial soldered connections created intermittent grounding failures, causing LED animations to cut out unexpectedly during operation.

Solution: Migrated circuit to breadboard implementation, which provided better contact reliability. Easier troubleshooting and component testing. Future iterations will use better shielded cables and more robust push-button mounting.

Problem 3: Vector Path Closure in VCarve

Multiple toolpath generation attempts failed because vector paths were not fully closed. Unclosed paths caused toolpath errors.

Solution: Implemented systematic checking of all path endpoints. Manually verified intersection points to ensure continuity. Created a verification checklist before initiating machine operation.

Results & Outcomes

Successfully created a personalized carved wooden heart (18" × 20") with integrated WS2812B LED lighting featuring multiple programmable animation sequences. The piece is suitable for display as a wall hanging or mantle decoration, with mounting options for both wall-mounted and freestanding use.

Lessons Learned

Early Project Initiation

Starting fabrication work well in advance of deadlines proved essential. Unforeseen technical issues (grounding problems, vector path errors, material constraints) would have created scheduling pressure if discovered near project completion.

Electrical System Design

Breadboarding should be considered as a primary prototyping method for circuits with grounding sensitivity. More robust mounting strategies for input devices prevent intermittent electrical failures.

Manufacturing Process Planning

Tool path verification in software saves significant time compared to troubleshooting failed cuts on material. Systematic vector closure checking should occur before any machine operation.

Design-for-Fabrication

Understanding material constraints early in the design phase—including standard stock dimensions and tool capabilities—prevents late-stage redesigns. Iterative CAD modeling allowed rapid adaptation when material size constraints were discovered.

Conclusion

This project successfully demonstrates the ability to integrate multiple manufacturing and electronics disciplines to create a personalized, functional art piece within makerspace constraints. The combination of CNC wood carving, 3D printing, and Arduino-based lighting control bridges traditional craftsmanship with modern electronics.

The finished piece serves as both a practical demonstration of fabrication techniques and a meaningful handmade gift combining technical achievement with personal significance. The project reinforced the importance of early planning, systematic problem-solving, and iterative design refinement.
